# ci-cd-python-demo
This project demonstrates a CI pipeline using GitHub Actions that:
-Automatically runs on push
-Sets up a Linux runner
-Installs Python 3.11
-Executes unit tests
-Reports pass/fail status

Technologies Used:
-Git
-GitHub Actions
-YAML
-Python
-Linux (Ubuntu runner)
-Unit testing (unittest)

What I Learned:
-CI/CD automation principles
-Workflow triggers
-Pipeline debugging
-Cloud-based test environments
